Como instalar

O Mypheed roda sobre o Node.js 24+ (gratuito) — os instaladores conferem e orientam se faltar.

  1. Baixe o Mypheed-Windows.zip e extraia para uma pasta sua (botão direito → Extrair Tudo…).
  2. Dê clique duplo em Instalar-Mypheed.bat — ele confere o Node.js, baixa as dependências e cria os atalhos com ícone no Desktop e no Menu Iniciar.
  3. Abra pelo ícone Mypheed: o servidor local sobe sozinho e a janela abre como um app.
Se o Windows avisar sobre "editor desconhecido" ao rodar o instalador: Mais informações → Executar assim mesmo. Se o Node.js não estiver instalado, o instalador abre a página oficial — instale a versão LTS e rode o instalador de novo.

Jeito recomendado — cole no Terminal (⌘+Espaço, digite “Terminal”):

curl -fsSL https://mypheed.myph.com.br/install-mac.sh | bash

Baixa o Mypheed para ~/Mypheed, instala as dependências e cria o app Mypheed no Launchpad. Rodar o mesmo comando de novo atualiza para a versão mais recente.

Ou manualmente:

  1. Baixe o Mypheed-macOS.tar.gz e dê clique duplo para extrair.
  2. No Terminal: bash ~/Downloads/Mypheed/Instalar-Mypheed.command
  3. Abra o Mypheed pelo Launchpad (arraste para o Dock se quiser).
Requer Node.js 24+ (o instalador abre a página se faltar). Na primeira exclusão de fotos o macOS pede permissão para o Node controlar o Finder — é a Lixeira; clique em Permitir. Se aparecer “desenvolvedor não identificado”, use clique direito → Abrir.

Perguntas frequentes

É grátis mesmo?

Sim. Sem assinatura, sem conta, sem limite de fotos. Os recursos de IA (estrelas e legendas) são os únicos que usam um serviço pago — e são opcionais, com a sua própria chave da Anthropic.

Minhas fotos saem do computador?

Não. O Mypheed é um servidor local que abre no seu navegador — culling, catálogo e exportação acontecem na sua máquina. Só os recursos de IA, quando você aciona, enviam previews pequenos para análise.

Funciona com o RAW da minha câmera?

Canon (CR2/CR3), Nikon (NEF), Sony (ARW), Fujifilm (RAF) e dezenas de outros formatos via ExifTool — qualquer RAW com preview embutido, além de JPEG, TIFF, PNG e vídeos. Pares RAW+JPEG viram um item só.

Por que preciso do Node.js?

O Node é o motor que roda o servidor local do Mypheed (leitura de previews, XMP, catálogo SQLite). É gratuito, de código aberto, e o instalador do Mypheed confere tudo para você.

Como atualizo o Mypheed?

Windows: baixe o zip novo e extraia por cima da pasta. macOS: rode de novo o comando de instalação. Suas configurações, catálogo e cache ficam fora da pasta do programa — nada se perde.

O culling aparece no Lightroom?

Sim: estrelas e cores são gravadas em XMP padrão (sidecar para RAW; embutido em JPEG/TIFF). O botão “Enviar para o Lightroom” abre a importação já filtrada na sua seleção.
